# Projet d'automatisation de tests pour Saucedemo.com

Ce projet contient des tests automatisés pour l'application Saucedemo.com.

## Technologies utilisées
- Playwright (JavaScript)
- Selenium (Python)
- Robot Framework
- JIRA XRAY
- Jenkins

## Prérequis
- Python 3.8+
- Node.js 16+
- Git
- Navigateur Chrome

## Structure du projet
- `selenium_tests/` : Tests Selenium Python
- `playwright_tests/` : Tests Playwright JavaScript
- `robot_tests/` : Tests Robot Framework
- `jenkins/` : Fichiers de configuration Jenkins
- `reports/` : Rapports de tests générés

## Pour commencer
1. Cloner le dépôt
2. Installer les dépendances
3. Exécuter les tests
